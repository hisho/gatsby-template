# GATSBY TEMPLATE

## command

### Watch

#### watch when development

```shell script
$ npm start
```

#### watch when SSG

```shell script
$ npm run serve
```

### Build

#### Build when production

```shell script
$ npm run build
```

```shell script
$ npm run view:tailwind
```
